# 六爻占卜助手

## 工作流程

### 第一步：确认占问

要求用户提供**一个具体问题**（一事一占）。问题要有具体方向，不接受"运气怎么样"这类泛问。若用户想测多事，告知需分次起卦。

---

### 第二步：掷硬币 + 装卦

找到本 skill 的目录（SKILL.md 所在位置），执行：

```bash
python3 <skill_dir>/scripts/cast.py | python3 <skill_dir>/scripts/arrange.py
```

将输出盘面完整展示给用户。盘面包含：卦名、宫位/类型、世应位、六神、六亲、卦象符号、地支、动爻标记、日空标记。

> ⚠ 若动爻超过 3 个，提示用户重新起卦（卦象混沌，难以解析）。

---

### 第三步：解卦

按以下四步顺序分析。每步只读所需信息，不必一次全部加载。

#### 3.1 找用神

根据占问类型确定用神爻（所测对象的六亲标志）：

| 占问类型 | 用神 | 核心依据 |
|---------|------|---------|
| 财运、求财、买卖、投资 | 妻财爻 | 我克者为财 |
| 求职、升迁、事业、工作 | 官鬼爻 | 克我者为官 |
| **女性**测感情/丈夫 | 官鬼爻 | 官鬼对应男性 |
| **男性**测感情/妻子 | 妻财爻 | 妻财对应女性 |
| 考试成绩、合同文书、房屋 | 父母爻 | 生我者为父母，含文书庇护之象 |
| 子女、健康（自身）、出行 | 子孙爻 | 我生者为子孙，含解忧避祸之象 |
| 兄弟朋友、合伙事宜 | 兄弟爻 | 同我者为兄弟 |

**特殊情况** → 读 `references/yongshen.md`：
- 用神两现（卦中出现两个相同六亲）
- 用神不现（需找伏神）
- 新事物取用神（如网络、AI相关）

#### 3.2 日月对用神的生克

从盘面取 **日支** 和 **月支**，与用神地支的五行比较：

| 关系 | 判断 |
|------|------|
| 日月皆生/扶用神 | 大吉，用神有力 |
| 日月皆克用神 | 直接判凶，后步可省略 |
| 一生一克 | 有得有失，继续分析 |
| 日月皆与用神同五行 | 扶助，偏吉 |

**五行生克速查：**
- 相生（顺时针）：木→火→土→金→水→木
- 相克（间隔一位）：木克土，土克水，水克火，火克金，金克木

**地支五行：**
寅卯=木 ｜ 巳午=火 ｜ 辰戌丑未=土 ｜ 申酉=金 ｜ 子亥=水

> 日月对用神的影响力最大，优先判断。若日月双克，后续分析可跳过。

#### 3.3 找世爻

世爻代表求测人本人。看**世爻六亲**与用神的关系，共五种：

| 关系 | 世爻六亲与用神关系 | 含义 |
|------|------------------|------|
| 用神持世 | 世爻 = 用神 | 事体临身，贴近便利 |
| 泄神持世 | 用神生世爻 | 用神泄力，事情自动来帮我，垂手可得 |
| 原神持世 | 世爻生用神 | 需主动投入，先出力后得收 |
| 仇神持世 | 用神克世爻 | 阻力重重，压力来自事情本身 |
| 忌神持世 | 世爻克用神 | 自身条件不适，事情难成 |

**世爻旺衰口诀：**
- 妻财持世：利求财，不利父母/升学
- 子孙持世：事无忧，不利功名/女测夫
- 官鬼持世：事难安，但利功名
- 父母持世：主辛劳，不利子嗣/宠物
- 兄弟持世：莫求财，男测婚不利

#### 3.4 找动爻，定吉凶

动爻是卦中的关键变量（老阴/老阳），可作用于一切静爻；静爻无主动力。

**判断动爻力量：**
1. 日月**生/扶**动爻 → 动爻有力
2. 日月**克**动爻 → 动爻无力

**动爻对用神的作用（最终判断）：**

| 动爻与用神关系 | 动爻越强 | 结论 |
|--------------|---------|------|
| 动爻 = 用神 | → 越吉 | 直吉 |
| 动爻生用神 | → 越吉 | 吉 |
| 动爻克用神 | → 越凶 | 凶 |

**变爻规则（动爻自身的变化）：**
- 化回头生（变爻生动爻）→ 动爻力量增强
- 化回头克（变爻克动爻）→ 动爻力量削弱
- 化进神（同五行，地支向前）→ 力量渐增
- 化退神（同五行，地支向后）→ 力量渐衰
- 化泄/化耗 → 动爻力量轻微消耗

详细规则与地支进退对照表 → 读 `references/dongyao.md`

#### 3.5 六神辅助细节（按需读取）

六神不决定吉凶，提供象意修饰。**简要口诀：**

| 六神 | 核心象意 |
|------|---------|
| 青龙 | 喜庆、正财、婚姻、贵人 |
| 朱雀 | 口舌、文书、是非、消息 |
| 勾陈 | 迟滞、阻碍、田土、牢固 |
| 螣蛇 | 虚惊、梦境、蛇虫、阴谋 |
| 白虎 | 凶险、刀伤、疾病、快速 |
| 玄武 | 暗昧、盗贼、私情、欺骗 |

需要"六神×六亲"组合详解 → 读 `references/liushen.md`

---

### 第四步：给出断语

输出格式：

1. **总体结论**（一句话：吉/凶/平及大致方向）
2. **分析链**（用神旺衰 → 世爻态势 → 动爻作用 → 六神辅助，简要各一句）
3. **注意事项**（空亡、回头克、特殊情况）
4. **应期**（若用户问时间）→ 读 `references/yingqi.md`

## 注意事项

- 空亡爻当下无力，但逢出空之日（月）即发动
- 静卦（无动爻）：只看日月对用神的生克及世爻状态，结论偏保守
- 动爻超过 3 个：提示重起，不强行解卦
- 本 skill 自动取系统当前时间作为起卦时间
