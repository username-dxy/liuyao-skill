# 动爻参考：变化规则详解 + 地支进退对照表

## 一、动爻基础

**动爻（老阴/老阳）** 是卦中的主动变量，能主动作用于其他爻；静爻无主动力，只受日月和动爻影响。

**动爻变化方向：**
- 老阳（────○）→ 变为阴爻（── ──）
- 老阴（──×──）→ 变为阳爻（─────）

动爻产生"变爻"（变卦中对应位置的爻），变爻对动爻本身有回馈影响。

---

## 二、变爻对动爻的五种作用

### 1. 化回头生（变爻生动爻）
- **定义：** 变爻五行生动爻五行
- **效果：** 动爻力量显著增强
- **断法：** 动爻作用于用神的力度加大（吉者更吉，凶者更凶）

**例：** 动爻地支为午（火），变爻地支为寅（木）→ 木生火，化回头生，动爻力量增强

---

### 2. 化回头克（变爻克动爻）
- **定义：** 变爻五行克动爻五行
- **效果：** 动爻力量显著削弱
- **断法：** 动爻作用于用神的力度减小（即使动爻本来生用神，力量也大打折扣）

**例：** 动爻地支为午（火），变爻地支为子（水）→ 水克火，化回头克，动爻力量受损

---

### 3. 化进神（动爻变为同五行更强地支）
- **定义：** 变爻与动爻同五行，且地支在进神序列中靠后（"进"）
- **效果：** 动爻力量渐增，越来越强
- **断法：** 事情发展有后劲，时间越久越有利/不利

**进神地支对：**

| 五行 | 动爻地支 → 进神地支 |
|------|-------------------|
| 木 | 寅 → 卯 |
| 火 | 午 → 巳（注：六爻以巳为火之进） |
| 金 | 申 → 酉 |
| 水 | 子 → 亥 |
| 土 | 辰 → 戌（或丑 → 未） |

> **口诀：** 寅化卯、午化巳、申化酉、子化亥、辰化戌（丑化未）为进神

---

### 4. 化退神（动爻变为同五行更弱地支）
- **定义：** 变爻与动爻同五行，且地支在退神序列中靠前（"退"）
- **效果：** 动爻力量渐衰，越来越弱
- **断法：** 事情发展后劲不足，开始还行，后来不济

**退神地支对（进神反向即退）：**

| 五行 | 动爻地支 → 退神地支 |
|------|-------------------|
| 木 | 卯 → 寅 |
| 火 | 巳 → 午 |
| 金 | 酉 → 申 |
| 水 | 亥 → 子 |
| 土 | 戌 → 辰（或未 → 丑） |

> **口诀：** 卯化寅、巳化午、酉化申、亥化子、戌化辰（未化丑）为退神

---

### 5. 化泄/化耗（变爻泄动爻）
- **定义：** 动爻五行生变爻五行（动爻"生出去"）
- **效果：** 动爻力量轻微消耗
- **断法：** 动爻作用力略减，但不像回头克那样严重
- **注意：** 与"化回头生"相反；化泄是动爻生变爻（动爻主动付出，自身消耗）

---

## 四、动爻综合判断流程

```
动爻地支 → 确认五行
    ↓
看日月是否生/克动爻
    ├─ 日月生动爻 → 动爻有力
    └─ 日月克动爻 → 动爻无力

    ↓
看变爻对动爻的作用
    ├─ 变爻生动爻（化回头生）→ 力量+
    ├─ 变爻克动爻（化回头克）→ 力量-
    ├─ 变爻同五行进神 → 力量渐增
    ├─ 变爻同五行退神 → 力量渐衰
    └─ 变爻受动爻所生（化泄）→ 力量微减

    ↓
综合判断动爻最终力量强弱
    ↓
看动爻与用神的关系
    ├─ 动爻生用神 → 动爻越强越吉
    ├─ 动爻克用神 → 动爻越强越凶
    └─ 动爻 = 用神（同六亲）→ 动爻越强越吉（直吉）
```

---

## 五、特殊注意

- **空亡动爻**：动爻若地支值日空，当下无力，但出空之日自然发动，届时再论吉凶
- **多动爻（4个以上）**：卦象混沌，按本 SKILL.md 主文建议重起卦
- **静卦（无动爻）**：仅看日月对用神的生克及世爻态势，结论偏保守，不宜大吉大凶论断
