# 六神参考：详细象意 + 六神×六亲速查

## 一、六神详细象意

### 青龙（木，吉神）
**核心性格：** 喜庆、正派、贵人、丰盛

| 领域 | 象意 |
|------|------|
| 人物 | 贵人、长官、有地位者、婚配佳偶 |
| 事件 | 喜事、升迁、宴席、好消息 |
| 钱财 | 正财进账、意外之喜 |
| 感情 | 姻缘正缘，婚配吉 |
| 健康 | 恢复顺利，少病痛 |
| 其他 | 美食美酒、欢聚庆典 |

---

### 朱雀（火，中性偏负）
**核心性格：** 口舌、文书、消息、是非

| 领域 | 象意 |
|------|------|
| 人物 | 口舌之人、文职人员、传话者 |
| 事件 | 合同文书、消息传递、官司口角 |
| 钱财 | 因口舌/文书得财，或因之破财 |
| 感情 | 多言多争，或通过媒人/文字缘起 |
| 健康 | 心肺、口腔、发热类疾病 |
| 其他 | 电话、信件、公告、宣传 |

---

### 勾陈（土，凶神偏迟滞）
**核心性格：** 迟滞、牵绊、田土、固守

| 领域 | 象意 |
|------|------|
| 人物 | 执拗之人、官府衙门、牢固守旧者 |
| 事件 | 拖延阻碍、纠纷缠绕、土地田产 |
| 钱财 | 来钱迟慢，财路不畅 |
| 感情 | 恋情黏滞，难断难舍 |
| 健康 | 脾胃、四肢、慢性病 |
| 其他 | 牢狱、羁绊、被人拖累 |

---

### 螣蛇（火，凶神偏虚惊）
**核心性格：** 虚惊、阴谋、梦境、蜿蜒

| 领域 | 象意 |
|------|------|
| 人物 | 心机之人、小人、表里不一者 |
| 事件 | 虚惊一场、阴谋算计、奇异之事 |
| 钱财 | 财路有诈，防骗防欺 |
| 感情 | 暗恋、秘密关系、心怀不轨 |
| 健康 | 心神不安、精神类、蛇虫咬伤 |
| 其他 | 噩梦、幻觉、捕风捉影 |

---

### 白虎（金，凶神最烈）
**核心性格：** 凶险、伤损、疾病、速度

| 领域 | 象意 |
|------|------|
| 人物 | 凶悍之人、医生、武人 |
| 事件 | 意外伤害、手术、官非、丧事 |
| 钱财 | 破财迅速，血光之灾 |
| 感情 | 争吵激烈，易分离 |
| 健康 | 外伤、手术、急症、骨骼、肺 |
| 其他 | 快速变化、车祸、意外 |

---

### 玄武（水，凶神偏暗昧）
**核心性格：** 暗昧、私情、盗贼、混沌

| 领域 | 象意 |
|------|------|
| 人物 | 小偷、出轨方、私下行事者 |
| 事件 | 失窃、背叛、私情暴露、暗箱操作 |
| 钱财 | 钱财不明、暗中破财 |
| 感情 | 私情出轨，或感情隐秘不公开 |
| 健康 | 肾脏、泌尿、性病、不明原因病症 |
| 其他 | 弄脏、混乱、见不得光之事 |

## 二、六神×六亲速查表

> 纵轴=六神，横轴=六亲。格内为核心断语（修饰用，不决定吉凶）。

|  | 父母爻 | 兄弟爻 | 子孙爻 | 妻财爻 | 官鬼爻 |
|--|--------|--------|--------|--------|--------|
| **青龙** | 贵人文书，合同顺利 | 贵人朋友，喜相帮 | 欢喜子孙，出行平安 | 正财丰厚，喜得财 | 贵官临门，升迁吉 |
| **朱雀** | 文书有口舌，合同有争议 | 朋友多口舌，是非缠绕 | 子孙好动，多言多语 | 因口得财，或财中有争 | 官司口舌，是非缠身 |
| **勾陈** | 文书迟滞，房产纠纷 | 朋友拖累，合伙有碍 | 子孙迟缓，出行受阻 | 财来迟慢，或涉土地产业 | 官职迟滞，升迁难速 |
| **螣蛇** | 文书有诈，合同有陷阱 | 朋友心机，防被算计 | 子孙多虑，虚惊不断 | 财有欺骗，防诈骗 | 暗中官非，防小人陷害 |
| **白虎** | 文书受损，房产有破 | 兄弟争财，朋友反目 | 子孙受伤，出行有险 | 财破迅速，血光财损 | 官威凶猛，疾病凶险 |
| **玄武** | 文书暗昧，合同有诈 | 朋友背后使绊，私下小动作 | 子孙行事隐秘，或宠物走失弄脏 | 财来不明，暗财或偷盗 | 官鬼暗中作祟，私情或暗病 |

## 三、特殊断法提示

**六神对宠物的象意（简易断）：**
- 青龙临子孙：玩得开心，被投喂，心情愉快
- 朱雀临子孙：遇到同伴，有交流互动，或被骂
- 勾陈临子孙：呆着不动，无目的溜达，或被人牵拉
- 螣蛇临子孙：四处游荡，机灵，有心眼，想着下次还来
- 白虎临子孙：顽皮打架，与其他动物争执，有受伤风险
- 玄武临子孙：到处乱跑，弄脏自己，有走失风险

**六神与日月关系：**
- 凶神（螣蛇、白虎、玄武）临旺爻：凶性放大
- 凶神临衰爻（被日月克）：凶性减弱，虚惊居多
- 吉神（青龙）临衰爻：好事难成，喜中有虑
