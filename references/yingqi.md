# 应期参考：应验时间推算

## 一、应期总诀

应期，指卦中所示事情最终应验（成事或败事）的时间节点。

**基本原则：**
1. 用神旺相 → 近期应验（日/周内）
2. 用神休囚 → 应期较远（月/季度后）
3. 用神空亡 → 出空之日/月应验
4. 有动爻作用 → 应在动爻地支对应的时日
5. 静卦无动爻 → 应在日月冲合用神之时

---

## 二、十二种应期方法

### 1. 用神动爻地支定应期
- **原则：** 动爻地支对应月份或日期即为应期
- **示例：** 用神为妻财动于子水爻 → 应在子月（11月）或子日
- **时间单位：** 近测用"日"，远测用"月"，大事用"年"

---

### 2. 冲开空亡定应期
- **原则：** 空亡爻在被冲之日/月出空，即为应期
- **地支相冲对：**

| 冲对 |
|------|
| 子 ↔ 午 |
| 丑 ↔ 未 |
| 寅 ↔ 申 |
| 卯 ↔ 酉 |
| 辰 ↔ 戌 |
| 巳 ↔ 亥 |

- **示例：** 用神子水空亡 → 午日/午月出空应验

---

### 3. 日月冲合用神定应期
- **冲用神之日** → 事情有变动，常为应期（尤其吉爻被冲或凶爻被冲散）
- **合用神之日** → 事情定局（合住则稳固，合绊则受阻）
- **地支相合（六合）：**

| 六合对 |
|--------|
| 子 ↔ 丑 |
| 寅 ↔ 亥 |
| 卯 ↔ 戌 |
| 辰 ↔ 酉 |
| 巳 ↔ 申 |
| 午 ↔ 未 |

---

### 4. 旺相休囚判远近
- **旺（当令）** → 近期，1-7天内
- **相（受生）** → 较近，2-4周内
- **休（生出去）** → 较远，1-3月
- **囚（同五行当令不是本身）** → 远，季度以上
- **死（被克）** → 极远或难应验

**月令旺相休囚表：**

| 月令（节气月） | 木旺 | 火旺 | 土旺 | 金旺 | 水旺 |
|--------------|------|------|------|------|------|
| 寅卯月（春）  | 旺  | 相  | 死  | 囚  | 休  |
| 巳午月（夏）  | 休  | 旺  | 相  | 死  | 囚  |
| 申酉月（秋）  | 死  | 囚  | 休  | 旺  | 相  |
| 亥子月（冬）  | 相  | 死  | 囚  | 休  | 旺  |
| 辰戌丑未月    | 囚  | 休  | 旺  | 相  | 死  |

---

### 5. 世爻被合之日定应期
- 世爻代表占者，世爻被日支相合之日 → 占者得到结果
- 常用于"何时得到消息/回复/批复"类占问

---

### 6. 用神受生之日定应期
- 用神五行被日支所生之日 → 用神得力，吉事应验
- **示例：** 用神为妻财（土），巳午日（火生土）为应期候选

---

### 7. 用神得令之月定应期
- 用神五行进入当令月份 → 应期
- **示例：** 用神为官鬼（金），申酉月金旺，为应期

---

### 8. 三合局完成之时定应期
- 动爻或用神为三合局中一员，待三合局完整之日为应期
- **三合局：**

| 三合局 | 地支组合 |
|--------|---------|
| 木局   | 亥卯未  |
| 火局   | 寅午戌  |
| 金局   | 巳酉丑  |
| 水局   | 申子辰  |

- **示例：** 动爻地支为亥（半合木局），待卯日/未日三合成局时应验

---

### 9. 爻位对应时间
- 初爻对应近期（1-2日）
- 二爻对应短期（3-7日）
- 三爻对应中期（半月）
- 四爻对应较远（1月）
- 五爻对应更远（3月）
- 上爻对应长远（半年以上）

> 此法为辅助参考，不可单独论断，需配合旺衰和动爻综合判断。

---

### 10. 飞爻伏爻出现之时
- 用神为伏神时，飞神被冲散之日 → 伏神出现，事情显现
- **冲飞神之日** = 应期

---

### 11. 忌神被制之时
- 当忌神（克用神的动爻）被日月或其他动爻克制之日 → 障碍解除，事情转机

---

### 12. 按五行速度判快慢

| 五行 | 速度象意 |
|------|---------|
| 火   | 最快，应期最近 |
| 木   | 较快 |
| 水   | 中等 |
| 金   | 较慢 |
| 土   | 最慢，应期最远 |

- 用神五行为火 → 应验快（日/周内）
- 用神五行为土 → 应验慢（月/季）
- 白虎临爻亦主速度快（含凶急之象）

---

## 三、综合应期判断口诀

> 动爻地支看应日，空亡出空是应期；
> 旺相近应休囚远，冲合用神定时机；
> 三合局成事始定，五行速度分快迟。
