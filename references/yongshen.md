# 用神参考：六亲象意 + 特殊情况处理

## 一、六亲完整象意表

| 六亲 | 五行关系 | 核心象意（人事） | 核心象意（事物） |
|------|---------|----------------|----------------|
| 父母爻 | 生我者 | 父母、长辈、上司、老师 | 文书、合同、房屋、学业、证件、车辆 |
| 兄弟爻 | 同我者 | 兄弟姐妹、朋友、同事、竞争者 | 劫财之象、阻力、平辈帮助 |
| 子孙爻 | 我生者 | 子女、晚辈、部下、医生 | 解忧避祸、健康、宠物、出行平安 |
| 妻财爻 | 我克者 | 妻子（男测）、财富、雇员 | 金钱、利益、货物、饮食、实物 |
| 官鬼爻 | 克我者 | 丈夫（女测）、官府、上司 | 功名、职位、压力、疾病、鬼魅 |

**五行关系速记：**
- 木→火→土→金→水→木（相生）
- 木克土，土克水，水克火，火克金，金克木（相克）

## 三、用神两现（卦中出现两个相同六亲）

**定义：** 六爻卦中，同一六亲在本卦中出现两次（含动变后的变卦爻）。

**处理原则：**

1. **以旺者为用** — 两爻皆静时，取日月生扶者为主用神
2. **以动者为用** — 一静一动时，优先取动爻为用神（动者有主动性）
3. **以近世者为用** — 旺衰相当时，世爻邻近的爻象意更切近占者

**实战口诀：** 动者优先，静则取旺，旺衰相当取近世。

## 四、用神不现（卦中无该六亲）

**定义：** 六爻中完全找不到所需六亲（含本卦和变卦）。

**处理方法：寻找伏神**

伏神藏于对宫（所属宫位的纯卦）各爻之下，称为"飞神伏神"关系。

步骤：
1. 确认用神在本卦六爻中确实不现
2. 找出本卦所属宫位（arrange.py输出中有"宫"字标注）
3. 该宫纯卦的对应位置爻即为飞神，其下伏藏的六亲即伏神
4. 判断飞神对伏神的作用：
   - 飞神生伏神 → 伏神有力，事有可为
   - 飞神克伏神 → 伏神受压，事难成
   - 飞神与伏神同五行 → 扶助，偏吉
   - 伏神克飞神（出伏） → 伏神力量强，可主事

**伏神口诀：** 飞来生合伏，伏神得出现；飞来克制伏，伏神难出头。

## 五、新事物的用神取法

现代占问中出现传统六爻未涉及的事物，参考以下对应：

| 现代事物 | 建议用神 | 理由 |
|---------|---------|------|
| 网络、互联网平台 | 父母爻 | 信息载体，含文书通讯之意 |
| AI、软件工具 | 子孙爻 | 辅助工具，类宠物/器械 |
| 股票投资 | 妻财爻 | 本质仍是求财 |
| 创业项目 | 官鬼爻（兼看妻财） | 事业功名 + 财利双看 |
| 出国、移民 | 子孙爻（兼看父母） | 出行安全 + 文件审批 |
| 健康体检结果 | 子孙爻（官鬼为病） | 子孙克官鬼，子孙旺则健康 |

> **灵活原则：** 用神不是死规定，理解"事物本质"后类比取用。有疑问时，先问自己：这件事对占者而言是"我在控制它"还是"它在约束我"？
